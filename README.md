# Odyssey

Step-based journey app: walk real steps, travel iconic trails with your traveler in 3D.

## Product vision

**Before you build or design, read the shared product context:**

→ **[docs/PRODUCT_VISION.md](docs/PRODUCT_VISION.md)**

That document covers trails, biomes, companions, collectibles (postcards, passport, curiosities), async social touches, and the overall expedition tone.

## Development

This repo is an **Expo** / **React Native** app with a **React Three Fiber** scene for the traveler and world.

```bash
npm install
npx expo start
```
